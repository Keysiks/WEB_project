Идея: бот для школьного расписания, можно заполнять дз, что-то вроде электронного журнала

Функции: 1. Школьное расписание, сделаю клавиатуру по дням, там можно будет заполнять дз и все такое
         2. Дополнение или изменения расписания через команды
         3. Выставлять оценки по предметам счиать среднне арифмитеческое и тд
         4. Функция возврата к предыдущему шагу
         5. Дз пишется заново на новый предмет, менять на новое в бд
